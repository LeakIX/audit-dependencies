### Related Ecosystem Vulnerabilities

While `@types/react-dom` itself has no vulnerabilities, the React ecosystem had
critical vulnerabilities in 2025 that users should be aware of:

#### CVE-2025-55182 (Critical - CVSS 10.0)

- **Affected:** React Server Components (react-server-dom-parcel,
  react-server-dom-turbopack, react-server-dom-webpack) versions 19.0.0,
  19.1.0, 19.1.1, 19.2.0
- **Fixed in:** React 19.0.3+, 19.1.4+, 19.2.3+
- **Description:** Unsafe deserialization vulnerability allowing
  unauthenticated remote code execution via malicious HTTP requests to Server
  Function endpoints
- **Note:** `@types/react-dom` is NOT affected directly, but should be updated
  alongside the core React packages for compatibility

#### Related CVEs

- CVE-2025-55184 (React Server Components)
- CVE-2025-67779 (React Server Components)
- CVE-2025-55183 (React Server Components)
- CVE-2025-66478 (Next.js)
