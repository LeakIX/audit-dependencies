# Security Audit: react v19.2.4

**Package:** [react](https://www.npmjs.com/package/react)
**Version:** 19.2.4
**Audit Date:** 2026-02-01
**License:** BSD-3-Clause

## Package Information

- **Description:** React is a JavaScript library for building user interfaces
- **Repository:** https://github.com/facebook/react
- **Homepage:** https://react.dev/
- **Maintainers:** Facebook (Meta)

## Security Status

**Overall Assessment:** SECURE (with caveats for React Server Components users)

The core `react` package (v19.2.4) itself has no known vulnerabilities. However,
applications using **React Server Components** must ensure they are using
patched versions of the related server-dom packages.

## Known Vulnerabilities

### Critical: CVE-2025-55182 - Remote Code Execution

| Field | Value |
|-------|-------|
| **CVE** | CVE-2025-55182 |
| **CVSS** | 10.0 (Critical) |
| **Type** | Unauthenticated Remote Code Execution |
| **Affected Packages** | react-server-dom-webpack, react-server-dom-parcel, react-server-dom-turbopack |
| **Affected Versions** | 19.0.0, 19.1.0, 19.1.1, 19.2.0 |
| **Fixed Versions** | 19.0.1, 19.1.2, 19.2.1 |
| **Disclosed** | 2025-12-03 |

**Description:** A critical vulnerability in React Server Components "Flight"
protocol allows attackers to achieve unauthenticated remote code execution
through insecure deserialization (CWE-502). Malicious HTTP requests can trigger
arbitrary code execution on the server.

**Advisory:** [GHSA-fv66-9v8q-g76r](https://github.com/facebook/react/security/advisories/GHSA-fv66-9v8q-g76r)

## Who Is Affected

You are affected if your application:

- Uses React Server Components
- Uses frameworks that support RSC: Next.js (App Router), React Router, Waku,
  @parcel/rsc, @vitejs/plugin-rsc, rwsdk

## Who Is NOT Affected

You are NOT affected if:

- Your React app does not use a server (client-side only)
- Your app does not use frameworks/bundlers supporting React Server Components
- You are using React 18.x or earlier

## Remediation

### For React Server Components Users

Upgrade to the latest patched versions:

```bash
npm install react@19.2.4 react-dom@19.2.4
npm install react-server-dom-webpack@19.2.4
# or react-server-dom-parcel@19.2.4 / react-server-dom-turbopack@19.2.4
```

### For Next.js Users

```bash
# For Next.js 14.x
npm install next@14.2.35

# For Next.js 15.0.x
npm install next@15.0.8

# For Next.js 15.1.x
npm install next@15.1.12

# For Next.js 15.2.x
npm install next@15.2.9

# For Next.js 16.0.x
npm install next@16.0.11
```

### Automated Fix Tool

Vercel provides an automated fix tool:

```bash
npx fix-react2shell-next
```
