## Related Package Vulnerabilities

While the core typescript package is secure, some packages in the TypeScript
ecosystem have had vulnerabilities:

### sequelize-typescript (CVE-2023-6293)

- **Affected Package:** sequelize-typescript (NOT typescript)
- **Type:** Prototype Pollution
- **Fixed In:** 2.1.6
- **Advisory:**
  [GHSA-7pvx-4585-hqww](https://github.com/advisories/GHSA-7pvx-4585-hqww)

### typescript-eslint (CVE-2021-35065)

- **Affected Package:** glob (dependency of typescript-eslint, NOT typescript)
- **Type:** Regular Expression Denial of Service
- **Status:** Low priority, duplicate issue

These vulnerabilities do **not** affect the typescript compiler package itself.

## Security Reporting

## Recommendations

1. **Safe to use:** TypeScript 5.9.3 is safe for production use
2. **Keep updated:** Subscribe to TypeScript releases for security patches
