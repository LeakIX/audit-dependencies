## Package Purpose

This package provides TypeScript type definitions and module aliases for
Docusaurus projects. It is primarily a development dependency that:

- Provides TypeScript types for Docusaurus modules
- Enables proper IDE support and type checking
- Contains no runtime code executed in production builds
