## Security Features

reqwest provides multiple security-relevant features:

- **TLS Backends**: Supports both native-tls and rustls for flexibility
- **Certificate Handling**: Custom CA certificates, client certificates
- **HTTPS Enforcement**: HTTPS-only mode available
- **Cookie Security**: Secure cookie jar implementation
- **Proxy Support**: SOCKS5 and HTTP proxy with authentication

## Dependencies

Key security-sensitive dependencies:

- `hyper` - HTTP implementation (same maintainer)
- `rustls` / `native-tls` - TLS implementations
- `http` - HTTP types
- `url` - URL parsing

All dependencies are well-maintained crates from the Rust ecosystem.
