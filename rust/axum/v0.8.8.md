## Vulnerabilities

### CVE-2022-3212 (RUSTSEC-2022-0055) - PATCHED

**Severity**: Medium (Denial of Service)
**Status**: Fixed in axum 0.5.16+ and 0.6.0-rc.2+

**Description**: No default limit on request bodies could allow attackers to
exhaust server memory via unbounded memory allocation.

**Affected Versions**:

- axum <= 0.5.15
- axum 0.6.0-rc.1

**Current Status**: Version 0.8.8 is NOT affected. The fix enforces a 2 MB
default request body limit.
