## Description

pytest-cov is a pytest plugin for measuring code coverage. It provides
coverage measurement functionality with several enhancements over standard
coverage tools including:

- Automatic management of coverage data files
- Support for detailed coverage contexts
- Compatibility with pytest-xdist for distributed testing
- Consistent pytest behavior
